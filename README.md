# 教学笔记 · (GitHub)

![Alt text](Figures/00Master/Multivariate-Statistics-Lists.png)

欢迎访问本仓库，包含了多元统计分析的模块化教学资源。每个专题的内容均采用 R Markdown 格式编写，涵盖了从基础统计到高级多元统计方法的各类主题。所有内容都将持续更新，您可以根据自己的需求自由使用。

## 课程概览

### 课程目录

| 专题编号 | 专题名称               | 课程简介                                         |
|----------|----------------------|------------------------------------------------|
|  01 | 统计基础               | 基本概念、变量类型、概率分布、置信区间          |

|  05 | 数据可视化             | 箱线图、云雨图、雷达图、相关性热图、词云图     |
|  06 | 因子分析               | 探索性因子分析、验证性因子分析                 |
|  07 | 信效度检验             | Alpha、重测信度、结构效度、聚敛效度、区分效度  |
|  08 | 共同方法偏差           | 程序控制、统计评估                             |
|  09 | 数据预处理             | 数据清洗、项目分析                             |

### 课程资源

本课程中的教学内容采用 R Markdown (.Rmd) 格式，图表、数据和链接在各模块之间共享。每个专题的内容都包含理论讲解、应用实例、可视化分析等。

### 软件及说明

- **SPSS**：统计及多元统计 + EFA + 信度检验
  - IBM SPSS Statistics (Version 29.0 或更高版本)
  
- **PROCESS**：中介 / 调节 / 有调节的中介

- **AMOS**：CFA + SEM + 效度检验
  - Windows-only；推荐插件： [Validity Master](Figures/00Master/Amos-Plugin.png)

- **R / G*Power**：数据可视化 + 样本量估计
